This Privacy Policy explains how {{ org_name }} ("we", "us") handles information when you use {{ app_name }} (the "App"). The App connects to your GitHub account via OAuth to access repository data you explicitly authorize.

What we access

  • GitHub account details necessary for OAuth and scopes you approve (e.g., public_repo, repo, workflow, read:org).
  • Repository metadata, commits, pull requests, workflow runs, artifacts, and file contents only as needed to fulfill your requests in the App.

How we use the data

We use GitHub data solely to perform the actions you ask for in the App (for example, listing commits or reading a file) and to improve reliability and security of the service. We do not sell your data or use it for advertising.

Token handling

The App does not operate its own token store. OAuth tokens are handled by the action platform that powers the App (OpenAI GPT Actions). We do not persist your GitHub access tokens on our servers. Tokens are used only to make API calls you initiate.

Retention

We do not maintain a separate database of your GitHub data. Information may appear in your chat history or logs associated with the App platform and be retained per that platform’s settings. You may revoke the App’s GitHub access at any time (see below).

Sharing

We do not share your GitHub data with third parties beyond the platforms required to provide the App (e.g., GitHub and OpenAI). We may disclose information if required by law or to protect our rights and users.

Security

All API requests use HTTPS. We recommend granting only the minimum scopes needed and limiting access to specific repositories where possible.

Your choices and rights

  • Revoke access: In GitHub, go to Settings → Applications → Authorized OAuth Apps and revoke the App.
  • Delete chat content: Manage or delete conversations within the platform hosting the App.
  • Data requests: Contact us at {{ contact_email }} for questions about this policy.
